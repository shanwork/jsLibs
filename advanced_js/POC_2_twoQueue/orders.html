Despatch Queue P.O.C.

  • Hi, this is a test of a despatch API I am writing..
  • On the left side is a set of executing jobs, each shown in green background.
  • Once each job gets over it gets transferred to the right, availabe queue, and displays in a pink background
  • Available jobs are restarted and get pushed to the executing queue
  • The continuous polling is for a default duration of minute
  • Once the duration is reached, jobs currently in progress run their course, while availble jobs remain unstarted.
  • To give a different duration enter a new value before clicking 'Statt Job'
