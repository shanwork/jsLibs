Sample Story Board

This showcases a storyboard template. Given a JSON object which is an array of sub-objects, the story is presented in sections or chapters, with one (at present) button per chapter

Each sub-object has the text content, styling for various elements in that view, button text, background image. This is a WIP iteratively evolving over time. The page navigation, content and styling are through CSS, core JavaScript (ES6 features), some (at this time) d3 APIs and my own MathsStatsAndMisc JavaScript framework
